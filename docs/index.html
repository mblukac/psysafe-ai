Building Safer AI Systems

Protect users and ensure ethical AI interactions with comprehensive psychological safety guardrails

PsySafe AI provides a modular, extensible framework for integrating psychological safety measures into AI applications. Our library helps developers create AI systems that can detect vulnerable users, prevent harm, and provide appropriate support while maintaining user privacy and ethical standards.

Getting Started

Start protecting your users in minutes

Installation

pip install psysafe-ai

Quick Example

from psysafe import GuardrailManager from psysafe.catalog import VulnerabilityDetectionGuardrail # Initialize the guardrail manager = GuardrailManager() vulnerability_guard = VulnerabilityDetectionGuardrail() manager.add_guardrail(vulnerability_guard) # Check user input result = manager.check("I feel so worthless and alone...") if result.requires_intervention: print(result.intervention_message)
