About PsySafe AI

PsySafe AI is an open-source collection of guardrail prompts designed to enforce ethical and safety constraints in AI language model applications. This project aims to imbue AI systems with fundamental principles that prioritize human safety, obedience to ethical instructions, and self-preservation of the system's integrity.

The repository provides a set of ready-to-use prompts and guidelines that developers can use as guardrails to improve AI safety and reliability of their existing or new AI applications.

Key Features

Ethical Compliance

Ensure AI assistants do not produce harmful content or violate privacy standards through carefully crafted guardrails.

User Well-being

Recognise user distress (e.g., mental health crises) to respond within ethical boundaries.

Framework Agnostic

Provide guardrails that work with any AI model or API (OpenAI, Anthropic Claude, DeepSeek, etc.) with minimal setup.

The Three Laws for AI Behavior

  1. Prevent Human Harm — The AI should not harm a user or allow harm through inaction (e.g., intervene if a user is in crisis).
  2. Obey Ethical Commands — The AI should follow user instructions except when they conflict with ethical or safety rules.
  3. Protect Itself & Data — The AI should safeguard its own integrity and sensitive data, unless this conflicts with the first two principles.
